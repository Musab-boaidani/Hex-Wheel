# Hex Wheel 🎨

This is a small side project I built in my free time, just for fun.

The idea was to create a **Wordle-like game**, but instead of guessing words, you try to **guess a HEX color**.

---

## 🧠 About

A simple interactive color game where:

* You guess a HEX color
* You get feedback to improve your next guess
* The goal is to match the correct color
